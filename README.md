# Requirements

* Node
* Bower

# Install and Test


Run the following

* npm install
* bower install
* npm run dev

# Config

configuration file is at `html/config.json`. modes are `auto` and `switch`.


# Keyboard actions

```
ArrowUp -> Previous # on switch mode
ArrowDown -> Next # on switch mode
ArrowRight -> Go into or select
ArrowLeft -> Go out
```
