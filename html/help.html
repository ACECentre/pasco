  1. Connect your switches. You need to map your switches as Space & Enter or W & S. Note if you want 4 swicth control Up & Down arrows will also work. Space and Enter allow you to Scan through the current selection and Enter select the item you want
  2. Configure your options. Enable on-screen navigation if you are a communication partner and are using this app to support someone. Automatic scanning or manual (Two-switch scanning). If you Select Automatic you will want to make the scan rate suitable for you.
  3. Choose a default tree of language or make your own. There are some example layouts in the software. You can use this but we will recommend that you adjust these for your own needs. There are a number of ways you can edit it. 1. In app - go to "Edit" in the menu, Select your item and either add, remove or edit the phrase. In iOS you can create a recorded message for the cue or main message. 2. Create your basic list in MindNode and export your mindmap as a markdown file. Import it. Done! 3. Create your structure in a text file and upload it in the config screen.
